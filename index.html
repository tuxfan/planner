Project Planner¶

Project Planner is a small Python planning tool for validating project task definitions, printing dependency-aware schedules, and exporting plans to Word and SVG files.

Installation¶

Install the package locally from the repository root:

python3 -m pip install -e .

Install documentation dependencies when you want to build this documentation:

python3 -m pip install -e ".[docs]"

Build the HTML documentation with:

python3 -m sphinx docs docs/_build/html
